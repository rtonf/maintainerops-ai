## Why this repository should qualify

MaintainerOps AI targets a core OSS bottleneck: maintainers must review PRs, triage issues, handle dependency/security alerts, and prepare releases with limited time. This public repo demonstrates that workflow end to end: npm package, GitHub Action, evals, SECURITY policy, Codex Security reports/fixes/full rescan, issue triage examples, real repo review packets, and successful Actions evidence. It is designed to improve security and code quality while keeping maintainers in control.

## How API credits will be used

API credits will be used only for repositories I own, maintain, or am authorized to administer. The project will generate structured maintainer packets for PR review, issue triage, security alert summarization, release readiness, and eval expansion. The API output is evidence-based and human-reviewed; it will not auto-merge, auto-close, publish releases, disclose vulnerabilities, or scan unauthorized repositories.
